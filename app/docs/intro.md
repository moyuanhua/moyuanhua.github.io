# 欢迎

欢迎来到 Murphy Blog 文档!

## 开始使用

这是一个基于 Docusaurus 构建的技术博客,内容同步自飞书知识库。

### 主要功能

- 📝 **双语支持**: 完整的中英文支持,自动语言检测
- 🔍 **本地搜索**: 支持中文分词的本地搜索功能
- 📱 **响应式设计**: 完美适配移动端和桌面端
- 🚀 **快速部署**: 基于 Cloudflare Pages 的全球 CDN 部署

## 文档结构

- [项目展示](/) - 查看我的项目
- [博客](/blog) - 阅读最新技术文章
- [关于我](/about) - 了解更多关于作者的信息

## 技术栈

本站点使用以下技术构建:

- **Docusaurus 3.9.2** - 静态站点生成器
- **React 18** - UI 框架
- **TypeScript** - 类型安全
- **feishu-pages** - 飞书内容同步
- **Cloudflare Pages** - 部署平台
