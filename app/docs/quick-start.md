# 快速开始

欢迎来到 Murphy Blog！这是一个基于 Docusaurus 和飞书知识库的现代化技术博客系统。

## 主要特性

- 📝 **双语支持**: 完整的中英文内容支持
- 🔄 **飞书集成**: 在飞书中编辑，自动同步到网站
- 🔍 **智能搜索**: 支持中文分词的全文搜索
- 📱 **响应式设计**: 完美适配各种设备
- 🚀 **快速部署**: Cloudflare Pages 全球 CDN

## 快速上手

### 1. 浏览文档
了解系统架构和核心功能，掌握项目的技术栈和设计理念。

### 3. 阅读博客
在[博客](/blog)中学习最新的技术文章和开发经验分享。

### 4. 了解更多
访问[关于页面](/about)了解作者信息和联系方式。

## 技术栈

| 技术 | 说明 |
|------|------|
| **React 19** | 最新的前端框架 |
| **TypeScript** | 类型安全的开发体验 |
| **Docusaurus 3.9** | 强大的文档站点生成器 |
| **飞书知识库** | 便捷的内容管理平台 |
| **Cloudflare Pages** | 快速的全球部署 |

## 核心功能

### 📝 内容管理
在飞书知识库中编辑内容，通过同步脚本自动更新到网站。支持 Markdown 格式，图片自动处理。

### 🌐 双语支持
中英文内容独立管理，自动语言检测，完整的 i18n 支持。

### 🔍 本地搜索
无需第三方服务，支持中文分词，实时搜索结果，关键词高亮。

### 🎨 主题定制
灵活的主题配置，支持亮色/暗色模式切换，响应式设计。
