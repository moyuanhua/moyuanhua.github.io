## 为什么选择 Docusaurus？

Docusaurus 是一个现代化的静态网站生成器，特别适合构建技术文档和博客：

- **开箱即用** - 零配置即可开始
- **React 驱动** - 使用 React 和 MDX
- **国际化** - 内置多语言支持
- **搜索功能** - 本地搜索或 Algolia
- **主题定制** - 灵活的主题系统

## 快速开始

```bash
npx create-docusaurus@latest my-website classic
cd my-website
npm start
```

## 核心功能

### 文档管理

Docusaurus 提供强大的文档管理功能，支持版本控制、侧边栏自动生成等。

### 博客系统

内置博客系统支持：
- 标签分类
- 作者信息
- RSS 订阅
- 归档页面

### 主题定制

通过 CSS 变量轻松定制主题：

```css
:root {
  --ifm-color-primary: #2e8555;
  --ifm-color-primary-dark: #29784c;
}
```

## 总结

Docusaurus 是构建技术博客的绝佳选择，值得一试！
